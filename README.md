# Rated List Creator

## Specifications
### Basic Requirements
- [x] Create, Add, Remove, Edit, Destroy
    - [x] Entries contains: Name, Date, Stars, Notess
- [x] Update ratings
- [x] Lists stores name, type of rating system
- [x] Save lists to file
- [x] Load lists on startup

### Optional
- [ ] Save rating history of entry (show previous ratings of the entry)
- [x] Create multiple lists and swapping between them
- [x] Export list to MD-file (or possibly HTML)
- [x] Watchlists
- [ ] Sorting lists
- [x] Change date of rating

### Very Optional
- [ ] Graphical interface
- [ ] Dymanic Fields (Can add new fields depending on what is interesting to the user)

## Submodules
- UI Handler
- List Handler
- List Exporter
- Integration tests
